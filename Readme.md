🏎️ SPECAR — Elite Hyper-Car Rental Node
SPECAR is a high-performance, immersive luxury vehicle rental platform. Designed with a brutalist "Cyber-Noir" aesthetic, the application utilizes React 19, Vite, and Framer Motion to deliver a premium, low-latency showroom experience for elite automotive enthusiasts.
📂 Project Structure
The codebase is organized into modular view layers and system nodes, centering around a core data telemetry file:
Plaintext
src/
├── components/         # Shared UI architecture (Nav, Footer, Modals)
├── pages/              # View Layers & System Nodes
│   ├── Index.jsx       # The Entry Node (Landing Page & Dashboard)
│   ├── Cars.jsx        # The Showroom (Fleet Listing & Filtering)
│   ├── CarsDetails.jsx # Vehicle Telemetry (Specs, Booking & Conditions)
│   ├── About.jsx       # Vision, Fleet Stats & Experience
│   ├── Contact.jsx     # Encrypted Transmission (Web3Forms Integration)
│   ├── Legal Terms.jsx # Master Service Agreement (01 // Access Protocols)
│   ├── Data Privacy.jsx# Privacy Protocols (The Biometric Shield)
│   └── Cookie Node.jsx # Node Taxonomy (System Architecture Policy)
├── Cars.json           # Centralized Fleet Telemetry (Database)
├── App.jsx             # Logical Routing (React Router v7)
└── main.jsx            # Application Ignition point
________________________________________
🛠 Tech Stack
•	Engine: React 19 + Vite
•	Aesthetics: Tailwind CSS 4 with a custom Cyan-on-Black palette.
•	Motion Architecture: Framer Motion 12 for staggered reveals and HUD-style transitions.
•	Showcase Engine: Swiper.js for high-fidelity car carousels.
•	Scheduling: React Datepicker for rental window selection.
•	Typography: Bricolage Grotesque.
________________________________________
✨ System Features
•	Showroom Telemetry: Dynamic filtering by vehicle class (Hyper, Elite, Stealth) and global pick-up locations (Mayfair HQ, Heathrow T5).
•	Request Access Flow: A specialized booking modal within CarsDetails that captures rental duration and triggers a "Request Logged" success state.
•	Node-Based Navigation: The site treats legal pages as "Data Nodes," maintaining a consistent futuristic UI even for policy documentation.
•	Transmission Protocols: Integrated Web3Forms in the Contact section for serverless, encrypted user inquiries.
•	Liability Tracking: Detailed insurance and liability class breakdowns for every vehicle in the fleet.

🚀 Deployment & Installation
1. Initialize the Node
Bash
git clone <your-repo-url>
cd specars-rental
npm install
2. Configuration
Ensure your Web3Forms access key is active in Contact.jsx to enable the transmission protocols:
JavaScript
formData.append("access_key", "YOUR_API_KEY");
3. Ignition
Bash
npm run dev
________________________________________
📜 System Scripts
•	npm run dev: Boot the local development environment.
•	npm run build: Compile the "Production-Ready" node.
•	npm run lint: Execute ESLint protocols to ensure code integrity.
